# 잘살아보세 - 부모용 가이드

## 사용 방법 (3단계)

### 1단계: 워크시트 출력
`worksheets/phase1/` 폴더에서 HTML 파일을 열고 **인쇄(Ctrl+P 또는 ⌘+P)**하세요.

### 2단계: 아이가 작성
- 아이에게 워크시트를 주고 손으로 작성하게 합니다
- 시간: 약 20-30분

### 3단계: AI 피드백 받기
1. 완성된 워크시트를 **사진으로 촬영**
2. ChatGPT, Claude, 또는 Gemini에 **사진 업로드**
3. 아래 프롬프트를 **복사해서 붙여넣기**

---

## AI 피드백 프롬프트 (복사해서 사용하세요)

### 📝 수준 평가 프롬프트
```
이 워크시트는 미국 5학년 수준의 Reading/Writing 연습입니다.
아이의 손글씨 답변 사진을 보고 다음을 평가해주세요:

1. 현재 수준 평가 (1-5점)
   - 5점: 완벽히 이해, 다음 단계 준비됨
   - 3-4점: 대체로 이해, 약간의 보완 필요
   - 1-2점: 추가 연습 필요

2. 잘한 점 (1-2개)
3. 개선할 점 (1개)
4. 다음 단계 권장:
   - 🟢 "다음 단계로 넘어가세요"
   - 🟡 "보완 워크시트 1장 더 연습 권장"
   - 🔴 "같은 유형 2-3장 더 연습 필요"

친절하고 격려하는 톤으로 한글 피드백해주세요.
아이에게 직접 전달할 수 있게 "~했구나", "~해보자" 형식으로 작성해주세요.
```

### 📝 보완 워크시트 생성 프롬프트
```
아이가 [워크시트 유형, 예: B-M-E 요약]에서 [어려워하는 부분, 예: Middle 갈등 설명]을 어려워합니다.

다음 조건으로 영어 보완 워크시트를 만들어주세요:
- 더 짧고 쉬운 읽기 지문 사용 (100-150 단어)
- [어려워하는 부분]에 집중하는 질문
- 힌트와 예시 제공
- 미국 5학년 수준 유지

출력 형식: 부모가 바로 인쇄할 수 있는 깔끔한 워크시트
```
